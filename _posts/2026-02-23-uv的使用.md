## 使用uv下载python

确认有哪些版本可供下载

使用 `uv python list` 列出所有版本

```powershell
uv python list
---------
# 得到类似于这样的输出
cpython-3.15.0a5-windows-x86_64-none                 <download available>
cpython-3.15.0a5+freethreaded-windows-x86_64-none    <download available>
cpython-3.14.2-windows-x86_64-none                   AppData\Roaming\uv\python\cpython-3.14.2-windows-x86_64-none\python.exe
...
```

如果要下载第三个版本：**cpython-3.14.2-windows-x86_64-none**

使用 `uv python install [python版本]`

```powershell
uv python install cpython-3.14.2-windows-x86_64-none
---------
Installed Python 3.14.2 in 5.59s
 + cpython-3.14.2-windows-x86_64-none (python3.14.exe)
```

这就下载了指定版本

## 使用uv初始化项目

使用 `uv init --python [版本]` 初始化项目

```powershell
# --python 是指定版本的可选参数
uv init --python 3.12.2
---------
Initialized project `dir_name`
```

使用 `uv venv --python [版本]` 初始化虚拟环境

```powershell
uv venv --python 3.12.2
---------
Using CPython 3.12.2
Creating virtual environment at: .venv
Activate with: .venv\Scripts\activate
```

初始环境搭建好了

## 从已有项目搭建环境

每个uv创建的项目都有两个文件：`uv.lock` 和 `pyproject.toml`

它们记录了项目的环境 ( ～'ω')～

将 `uv.lock` 和 `pyproject.toml` 克隆到项目文件夹

使用 `uv sync` 从已有项目搭建环境

```powershell
uv sync
---------

Creating virtual environment at: .venv
Resolved 8 packages in 1ms

 + et-xmlfile==2.0.0
```
